## About MyVillager

## Description
MyVillager is a simple plugin allowing the players to claim villagers as their own.  
You can set the claiming price of villager. (Need Vault installed)  
After claiming the villager(s), other players are not able to trade and hurt them.  
The plugin can also prevent the claimed villagers getting hurt from anvil, fall and fire damage etc.  
The owners can also set up permission group for trusting players.

## Version Support
