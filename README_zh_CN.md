## 关于 MyVillager

## 简介
这是一款轻量的插件, 使玩家可以认领村民  
该功能非常适合无领地的纯生存/生电服务器  
在认领了村民后, 被认领的村民无法被未授权玩家攻击及交易  
被认领村民的主人可以设置权限组以授权部分玩家能够与该村民交易  
被认领的村民也拥有多方位的保护, 例如来自摔落、窒息、僵尸、掠夺者的伤害等  
支持经济插件，可设置认领的价格（需要前置Vault）
